Obsidian AI 업무 Wiki의 맥락 오염을 검사하는 규칙입니다.

## Lint 검사 항목

| 항목 | 확인할 것 |
| --- | --- |
| Raw 불변성 | `raw/`, `10_RAW_SOURCE/` 원문이 임의 수정되지 않았는가 |
| 출처 | Wiki/Core의 중요한 주장에 원문 또는 근거 링크가 있는가 |
| 결정 기록 | 결정 내용, 이유, 날짜, 결정자가 남아 있는가 |
| 추측 분리 | AI 추론이 확정 사실처럼 저장되지 않았는가 |
| 오래된 규칙 | 현재 구조와 맞지 않는 예전 규칙이 남아 있지 않은가 |
| 인덱스 | 새 핵심 문서가 `index.md` 또는 `INDEX.md`에 연결되었는가 |
| 로그 | 의미 있는 작업이 `log.md` 또는 `LOG.md`에 기록되었는가 |
| 중복 | 같은 내용이 여러 문서에 충돌되게 저장되지 않았는가 |

## Lint 결과 형식

```text
Lint 결과:
- 통과:
- 경고:
- 수정 필요:
- 확인 필요:
```

## 자동 수정 가능

- 누락된 내부 링크 추가
- 로그에 작업 요약 추가
- 인덱스에 새 문서 링크 추가
- 명백한 오타 수정

## 사용자 확인 필요

- Raw Source 수정 여부
- 오래된 결정 폐기
- 서로 충돌하는 정책 중 우선순위 선택
- 프로젝트 범위 변경
- 결정자/근거가 없는 결정을 확정 처리하는 일
