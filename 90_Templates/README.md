# Developer Obsidian Templates

개발 공부, 블로그 초안, 코딩테스트, 프로젝트 정리, 에러 기록을 위한 Obsidian 템플릿 모음입니다.

## 사용 방법

1. 이 압축 파일을 다운로드합니다.
2. 압축을 풉니다.
3. `.md` 파일들을 Obsidian Vault의 `90_Templates` 폴더에 복사합니다.
4. Obsidian에서 `Settings → Core plugins → Templates`를 켭니다.
5. `Settings → Templates → Template folder location`을 `90_Templates`로 설정합니다.
6. 새 노트에서 `Insert template` 명령으로 원하는 템플릿을 삽입합니다.
