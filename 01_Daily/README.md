`01_Daily/`는 일일 기록, TIL, 회고, 짧은 작업 로그를 관리하는 공간입니다.

## 사용 규칙

- 날짜 문서는 `YYYY-MM-DD-title.md` 형식을 사용합니다.
- 프로젝트 작업을 했다면 해당 프로젝트의 `00_SCHEMA/LOG.md`에도 요약을 남깁니다.
- 일일 기록에서 나온 결정 사항은 프로젝트의 `30_DECISIONS/` 또는 관련 학습 노트로 분리합니다.
