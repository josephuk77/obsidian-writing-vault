### 머신러닝이란
머신러닝은 데이터셋으로 표현된 현상을 근사하는 모델 f(.)를 학습하는 방법

![](https://velog.velcdn.com/images/josephuk77/post/3c3ba665-7c0d-4246-a7c7-0d5fa14d48e9/image.png)

#### 머신러닝을 사용하는 경우
- 문제를 해결하기 위해 필요한 알고리즘의 복잡도가 높아, 사람이 만들어 내는 룰만으로는 풀기 힘든 문제일 경우
- 입력 차원 또는 출력 차원이 너무 고차원인경우
- 머신러닝을 사용하는 경우의 예
	
    - 글자 인식(OCR)
    - Question Answering
    - 감정 분석
    - 이미지 분류
    - 이미지 생성

#### 머신러닝을 사용하면 비효율적인 경우
- 문제를 해결하기 위한 오리고리즘이 명확한 경우
- 간단한 heuristic 알고리즘으로도 충분히 서비스 요구사항에 충족되는 알고리즘이 있는 경우
- 머신러닝을 사용하면 비효율적인 경우의 예
	
    - 단순 계산기
    - 시간 계산
    - 환전 수수로 계산
    - 호텔 저가순 정렬
    - 상품 이름순 정렬
    - 미팅을 위한 빈시간 찾기

#### 머신러닝의 학습 방법
- Supervised Learning
	
    - 입력 값에 대한 출력 값을 매핑하는 모델을 학습하는 방법
    - 분류(Classification), 회귀(Regression)
    - ex)이미지 분류, 주식 데이터 미래 예측
- Unsupervised Learning
	
    - 데이터의 기저에 있는 패턴을 찾는 모델 학습 방법
    - 군집화(Clustering),차원 축소(Dimensional reduction)
    - ex)추천 서비스
- Self-supervised Learning
	
    - Predictive learning을 하여 데이터의 기처에 있는 원리, 패턴을 학습하는 모델 학습 방법
    - word2vec, BERT, GPT-3
    - ex)생성형 AI 모델
- Reinforcement Learning
	
    - 시뮬레이션 할 수 있는 환경이 주어진 상황에서 모델이 환경과 상호작용하며 보상을 얻거나 처벌을 받으며 모델이 환경에서 보상을 최대한 많이 획득하도록 학습하는 방법
    - Alphaho, self-driving car, robotics manipulation
