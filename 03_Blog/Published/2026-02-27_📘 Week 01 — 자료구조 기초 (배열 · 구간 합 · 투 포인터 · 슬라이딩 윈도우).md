# 📘 Week 01 — 자료구조 기초 (배열 · 구간 합 · 투 포인터 · 슬라이딩 윈도우)

# 배열과 리스트 그리고 벡터

## 배열의 특징

- 인덱스로 즉시 접근 가능 (O(1))
- 중간 삽입/삭제는 비효율적
- 크기 고정
- 코딩 테스트에서 가장 많이 사용

## 리스트의 특징

- 인덱스 없음 (순차 접근)
- 삽입/삭제 빠름
- 크기 가변
- 구조가 비교적 복잡

## 벡터(Vector)의 특징

- 동적 배열
- push_back 빠름
- 중간 삽입/삭제는 느림
- 인덱스 접근 가능

# ✅ 문제 1 — 숫자의 합

## 핵심 아이디어
문자열로 입력받아 `'0'`을 빼서 정수 변환

## 정답 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N;
    string numbers;
    int sum = 0;

    cin >> N;
    cin >> numbers;

    for(int i = 0; i < N; i++) {
        sum += numbers[i] - '0';
    }

    cout << sum << "\n";
}
```

# ✅ 문제 2 — 평균

## 핵심 아이디어
최댓값 구한 뒤  
`sum * 100.0 / max / N`

## 정답 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    int N;
    cin >> N;

    int A[1000];

    for(int i = 0; i < N; i++)
        cin >> A[i];

    long sum = 0;
    long maxValue = 0;

    for(int i = 0; i < N; i++) {
        if(A[i] > maxValue)
            maxValue = A[i];
        sum += A[i];
    }

    double result = sum * 100.0 / maxValue / N;
    cout << result << "\n";
}
```

# 📌 구간 합 (Prefix Sum)

## 1차원 구간 합 공식

```
S[i] = S[i-1] + A[i]
구간합 = S[j] - S[i-1]
```

# ✅ 문제 3 — 구간 합 구하기 1

## 핵심 아이디어
누적합 배열 사용

## 정답 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N, M;
    cin >> N >> M;

    long S[100001] = {0};

    for(int i = 1; i <= N; i++) {
        int num;
        cin >> num;
        S[i] = S[i-1] + num;
    }

    for(int i = 0; i < M; i++) {
        int a, b;
        cin >> a >> b;
        cout << S[b] - S[a-1] << "\n";
    }
}
```

# ✅ 문제 4 — 구간 합 구하기 2 (2차원)

## 핵심 아이디어
2차원 누적합

## 정답 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N, M;
    cin >> N >> M;

    long S[1025][1025] = {0};

    for(int i = 1; i <= N; i++) {
        for(int j = 1; j <= N; j++) {
            int num;
            cin >> num;
            S[i][j] = S[i][j-1] + S[i-1][j] - S[i-1][j-1] + num;
        }
    }

    for(int i = 0; i < M; i++) {
        int x1,y1,x2,y2;
        cin >> x1 >> y1 >> x2 >> y2;

        cout << S[x2][y2]
             - S[x1-1][y2]
             - S[x2][y1-1]
             + S[x1-1][y1-1] << "\n";
    }
}
```

# ✅ 문제 5 — 나머지 합

## 핵심 아이디어
누적합 % M 값이 같은 개수 조합

## 정답 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    long N, M;
    cin >> N >> M;

    long long S = 0;
    long long count[1001] = {0};
    long long result = 0;

    count[0] = 1;

    for(int i = 0; i < N; i++) {
        long num;
        cin >> num;
        S = (S + num) % M;
        result += count[S];
        count[S]++;
    }

    cout << result;
}
```

# ✅ 문제 6 — 수들의 합 5

## 정답 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    int N;
    cin >> N;

    int start = 1, end = 1;
    int sum = 1;
    int count = 1;

    while(end != N) {
        if(sum == N) {
            count++;
            end++;
            sum += end;
        }
        else if(sum > N) {
            sum -= start;
            start++;
        }
        else {
            end++;
            sum += end;
        }
    }

    cout << count;
}
```

# ✅ 문제 7 — 주몽

## 정답 코드

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {

    int N, M;
    cin >> N >> M;

    int A[15000];

    for(int i = 0; i < N; i++)
        cin >> A[i];

    sort(A, A + N);

    int i = 0, j = N - 1;
    int count = 0;

    while(i < j) {
        if(A[i] + A[j] == M) {
            count++;
            i++;
            j--;
        }
        else if(A[i] + A[j] < M)
            i++;
        else
            j--;
    }

    cout << count;
}
```

# ✅ 문제 8 — 좋다

## 정답 코드

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {

    int N;
    cin >> N;

    long A[2000];

    for(int i = 0; i < N; i++)
        cin >> A[i];

    sort(A, A + N);

    int result = 0;

    for(int k = 0; k < N; k++) {

        int i = 0;
        int j = N - 1;

        while(i < j) {

            if(i == k) { i++; continue; }
            if(j == k) { j--; continue; }

            long sum = A[i] + A[j];

            if(sum == A[k]) {
                result++;
                break;
            }
            else if(sum < A[k])
                i++;
            else
                j--;
        }
    }

    cout << result;
}
```

# ✅ 문제 9 — DNA 비밀번호 (12891)

## 정답 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int checkArr[4];
int myArr[4];
int checkSecret = 0;

void Add(char c) {
    switch(c) {
        case 'A': myArr[0]++; if(myArr[0] == checkArr[0]) checkSecret++; break;
        case 'C': myArr[1]++; if(myArr[1] == checkArr[1]) checkSecret++; break;
        case 'G': myArr[2]++; if(myArr[2] == checkArr[2]) checkSecret++; break;
        case 'T': myArr[3]++; if(myArr[3] == checkArr[3]) checkSecret++; break;
    }
}

void Remove(char c) {
    switch(c) {
        case 'A': if(myArr[0] == checkArr[0]) checkSecret--; myArr[0]--; break;
        case 'C': if(myArr[1] == checkArr[1]) checkSecret--; myArr[1]--; break;
        case 'G': if(myArr[2] == checkArr[2]) checkSecret--; myArr[2]--; break;
        case 'T': if(myArr[3] == checkArr[3]) checkSecret--; myArr[3]--; break;
    }
}

int main() {

    int S, P;
    cin >> S >> P;

    string str;
    cin >> str;

    for(int i = 0; i < 4; i++) {
        cin >> checkArr[i];
        if(checkArr[i] == 0) checkSecret++;
    }

    int result = 0;

    for(int i = 0; i < P; i++)
        Add(str[i]);

    if(checkSecret == 4) result++;

    for(int i = P; i < S; i++) {
        int j = i - P;
        Add(str[i]);
        Remove(str[j]);
        if(checkSecret == 4) result++;
    }

    cout << result;
}
```

# ✅ 문제 10 — 최솟값 찾기 (11003)

## 정답 코드

```cpp
#include <iostream>
#include <deque>
using namespace std;

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N, L;
    cin >> N >> L;

    deque<pair<int,int>> dq;

    for(int i = 0; i < N; i++) {

        int num;
        cin >> num;

        while(!dq.empty() && dq.back().first > num)
            dq.pop_back();

        dq.push_back({num, i});

        if(dq.front().second <= i - L)
            dq.pop_front();

        cout << dq.front().first << " ";
    }
}
```
