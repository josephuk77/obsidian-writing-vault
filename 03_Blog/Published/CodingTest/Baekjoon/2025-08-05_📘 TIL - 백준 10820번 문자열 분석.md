# 📘 TIL - 백준 10820번: 문자열 분석

🔍 문제 요약
여러 개의 문자열이 주어졌을 때, 각 문자열마다 포함된 문자들을 다음 네 가지로 분류해서 개수를 출력하는 문제.

소문자 (a~z)

대문자 (A~Z)

숫자 (0~9)

공백 (space)

✨ 핵심 아이디어
📌 문자열 순회 + 문자 분류
문자열을 한 글자씩 순회하면서 Character 클래스의 메서드를 활용하여 문자 유형을 판별한다.

문자열마다 소문자, 대문자, 숫자, 공백 개수를 각각 카운트한다.

입력이 주어지는 동안 계속 반복해서 처리 (EOF 방식)

🧠 배운 점
Character 클래스의 메서드

Character.isLowerCase(char) → 소문자인지 판별

Character.isUpperCase(char) → 대문자인지 판별

Character.isDigit(char) → 숫자인지 판별

char == ' ' → 공백인지 판별

EOF 입력 처리

while ((line = br.readLine()) != null)을 통해 EOF까지 처리할 수 있음

문자열을 배열처럼 순회

toCharArray() 또는 charAt(i) 방식 모두 활용 가능

printf 포맷팅 출력

%d %d %d %d\n 형식으로 깔끔하게 출력 가능

🧑‍💻 핵심 코드 요약
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
String input;
while ((input = br.readLine()) != null) {
    int lower = 0, upper = 0, digit = 0, space = 0;
    for (char ch : input.toCharArray()) {
        if (Character.isLowerCase(ch)) lower++;
        else if (Character.isUpperCase(ch)) upper++;
        else if (Character.isDigit(ch)) digit++;
        else if (ch == ' ') space++;
    }
    System.out.printf("%d %d %d %d%n", lower, upper, digit, space);
}
```
🧮 시간/공간 복잡도
항목	분석
시간 복잡도	O(N * L) → N = 줄 수, L = 문자열 길이
공간 복잡도	O(1) → 정수형 변수 4개만 사용

🎯 느낀 점
문자열 처리 문제에서는 자바의 Character 유틸리티 메서드를 적극 활용하는 습관이 중요하다.

EOF 입력을 처리하는 구조는 백준에서 자주 등장하므로 익숙해질 필요가 있다.

단순한 문제더라도 코드 가독성과 효율성을 함께 고려해야 한다.

입력 조건이 작으면 복잡한 자료구조 없이도 단순 순회 + 조건문으로 충분히 해결할 수 있다.
