# 백준 1157번 — 단어 공부 (Java)

## 문제 설명

알파벳 대문자와 소문자로 이루어진 단어가 주어졌을 때
이 단어에서 **가장 많이 사용된 알파벳**을 출력하는 문제이다.

단, 다음 조건을 만족해야 한다.

* **대문자와 소문자를 구분하지 않는다**
* 가장 많이 사용된 알파벳이 **여러 개 존재하면 `?` 출력**

---

## 입력 조건

* 알파벳 대문자와 소문자로 이루어진 문자열
* 문자열 길이: **최대 1,000,000**

---

## 출력

가장 많이 사용된 알파벳을 **대문자로 출력**

단,
```text
가장 많이 사용된 알파벳이 여러 개이면
?
```
를 출력한다.

---

## 입력 예시
```text
Mississipi
```
## 출력 예시
```text
?
```
---

## 문제 해결 아이디어

문제의 핵심은 **알파벳의 개수를 세는 것**이다.

알파벳은 총 **26개**이므로
다음과 같은 배열을 사용할 수 있다.
```text
int[26]
```
각 인덱스는 다음 의미를 가진다.

| 인덱스 | 알파벳 |
| --- | --- |
| 0   | A   |
| 1   | B   |
| 2   | C   |
| ... | ... |
| 25  | Z   |

문자를 숫자로 변환하는 방법
```text
문자 - 'A'
또는
문자 - 'a'
```
예
```text
'A' - 'A' = 0
'B' - 'A' = 1
'C' - 'A' = 2
```
이 방식을 사용하면 **각 알파벳의 개수를 쉽게 셀 수 있다.**

---

# 강의 풀이

강의 풀이에서는 다음과 같은 방식으로 문제를 해결한다.

1. 문자열을 **대문자로 변환**
2. 알파벳 개수 배열 생성
3. 가장 큰 값 찾기
4. 최대값이 여러 개인지 검사

## 코드
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String input = sc.nextLine();
        String upper = input.toUpperCase();
        int[] arr = getAlphabetCount(upper);

        int index = findMaxIndex(arr);

        if(index == '?')
            System.out.print("?");
        else 
            System.out.print((char)(index+'A'));
    }

    private static int findMaxIndex(int[] arr) {
        int max = 0;
        int index = 0;
        int same = -1;
        for(int i = 0; i < arr.length; i++){
            if(arr[i] > max){
                max = arr[i];
                index = i;
            }
        }
        for (int j : arr) {
            if (j == max) {
                same++;
            }
        }
        if(same == 0)
            return index;
        else
            return '?';
    }

    public static int[] getAlphabetCount(String str){
        int[] arr = new int[26];
        for(char a : str.toCharArray()){
            arr[a - 'A']++;
        }
        return arr;
    }
}
```

# 내가 작성한 풀이

내 풀이는 다음 방식으로 구현했다.

1. 문자열을 **소문자로 변환**
2. 알파벳 개수 배열 생성
3. 최대값과 인덱스를 동시에 탐색
4. 동일한 값이 발견되면 `-1`로 처리

## 코드
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        char[] input = sc.nextLine().toLowerCase().toCharArray();
        int[] count = new int[26];
        int result = 0, tmp = 0;
        
        for(char c : input) {
            count[c-'a']++;
        }

        for(int i = 0; i < count.length; i++) {
            if(tmp < count[i]) {
                result = i;
                tmp = count[i];
            }else if(tmp == count[i]) {
                result = -1;
            }
        }

        if(result == -1)
            System.out.println("?");
        else
            System.out.println((char)('A' + result));
        
    }
}
```

# 핵심 개념

## 1. 알파벳 카운트 배열

알파벳은 총 26개이므로
```text
int[26]
```
배열을 사용하면
각 문자 개수를 빠르게 저장할 수 있다.

---

## 2. 문자 → 인덱스 변환

문자를 배열 인덱스로 변환하는 방식
```text
문자 - 'a'
또는
문자 - 'A'
```
예
```text
'a' - 'a' = 0
'b' - 'a' = 1
```
---

## 3. 대소문자 통일

문제에서 대소문자를 구분하지 않기 때문에
먼저 문자열을 한쪽으로 통일한다.
```text
toUpperCase()
또는
toLowerCase()
```

1. **알파벳 개수를 배열로 저장한다.**
