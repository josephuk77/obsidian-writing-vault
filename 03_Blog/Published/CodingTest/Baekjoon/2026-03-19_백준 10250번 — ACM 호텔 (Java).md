# 백준 10250번 — ACM 호텔 (Java)

## 문제 설명

ACM 호텔은 `H`층, 각 층마다 `W`개의 방이 있는 구조이다.
손님은 도착 순서대로 방을 배정받으며, 다음 기준을 따른다.
```text
1. 엘리베이터와 가까운 방을 우선 배정
2. 같은 거리라면 더 높은 층 선호
```
즉, 방 배정 순서는 다음과 같다.
```text
101 → 201 → 301 → ... → H01 → 102 → 202 → ...
```
즉 **층을 먼저 채우고, 그 다음 호수를 증가**시키는 방식이다.

---

## 입력

첫째 줄
```text
T
```
테스트 케이스 개수

각 테스트 케이스
```text
H W N
```
* H : 층 수
* W : 각 층의 방 수
* N : N번째 손님

조건
```text
1 ≤ H, W ≤ 99
1 ≤ N ≤ H × W
```
---

## 출력

N번째 손님이 배정받을 방 번호 출력

형식
```text
층 + 호수 (2자리)
```
예
```text
402
1203
```
---

## 입력 예시
```text
2
6 12 10
30 50 72
```
## 출력 예시
```text
402
1203
```

# 문제 해결 아이디어

이 문제의 핵심은 다음이다.
```text
층을 먼저 채운다
```
즉

* 층 = N % H
* 호수 = N / H

로 계산할 수 있다.

---

## 예시
```text
H = 6, N = 10
```
층
```text
10 % 6 = 4
```
호수
```text
10 / 6 = 1 → +1 → 2
```
결과
```text
402
```

# 내가 작성한 풀이

나는 나머지와 몫을 이용해 직접 계산하고,
문자열로 결과를 만들어 출력했다.

## 코드
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int T = sc.nextInt();

        String ans = "";

        for(int i = 0; i < T; i++) {

            int H = sc.nextInt();
            int W = sc.nextInt();
            int N = sc.nextInt();

            int hight = N % H == 0 ? H : N % H;

            int width = N % H == 0 ? N / H : N / H + 1;

            ans += hight;

            if(width < 10)
                ans += ("0" + width);
            else
                ans += width;

            ans += "\n";
        }

        System.out.println(ans);
    }
}
```

# 강의 풀이

강의에서는 동일한 로직을 더 간단하게 표현했다.

## 코드
```java
import java.util.Scanner;

class Main
{
    public static void main (String[] args) {

        Scanner sc = new Scanner(System.in);

        int T = sc.nextInt();

        while (T-- > 0) {

            int H = sc.nextInt();
            int W = sc.nextInt();
            int N = sc.nextInt();

            int floor = ((N - 1) % H) + 1;
            int number = (N - 1) / H + 1;

            System.out.printf("%d%02d\n", floor, number);
        }
    }
}
```
---

# 두 풀이 방식 비교

| 방식    | 특징               |
| ----- | ---------------- |
| 내 풀이  | 조건문으로 나머지 처리     |
| 강의 풀이 | `(N-1)`로 간단하게 처리 |

---

# 핵심 차이

## 내 코드
```java
N % H == 0 ? H : N % H
```
```java
N % H == 0 ? N/H : N/H + 1
```
→ 나머지가 0일 때 따로 처리 필요

---

## 강의 코드
```java
(N - 1) % H + 1
(N - 1) / H + 1
```
→ 모든 경우를 하나의 식으로 처리 가능

---

# 핵심 개념

## 1. 층 계산
```text
층 = ((N - 1) % H) + 1
```
---

## 2. 호수 계산
```text
호수 = ((N - 1) / H) + 1
```
---

## 3. 출력 포맷
```java
System.out.printf("%d%02d\n", floor, number);
```
* `%02d` → 두 자리로 출력 (앞을 0으로 채움)

# 정리

이 문제는 처음 보면 복잡해 보이지만,
실제로는 **나머지와 몫을 이용한 간단한 수학 문제**였다.

처음에는 나머지가 0인 경우를 따로 처리했지만,
강의 풀이처럼 `(N-1)`을 활용하면 모든 경우를 하나의 식으로 처리할 수 있다.

즉,
```text
(N-1) 활용 → 조건문 제거 → 코드 단순화
```
이 문제의 핵심 포인트였다.
