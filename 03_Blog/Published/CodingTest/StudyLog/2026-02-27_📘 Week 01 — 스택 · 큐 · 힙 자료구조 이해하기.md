# 📘 Week 01 — 스택 · 큐 · 힙 자료구조 이해하기

# 스택(Stack)과 큐(Queue)

## 스택(Stack)

스택은 삽입과 삭제 연산이 **후입선출(LIFO : Last In First Out)** 구조로 이루어지는 자료구조이다.

### 스택 용어

#### 위치

- top : 가장 위에 있는 데이터 위치

#### 연산

- push : 데이터 삽입
- pop : 데이터 삭제
- top : 가장 위 데이터 확인

---

## 큐(Queue)

큐는 삽입과 삭제 연산이 **선입선출(FIFO : First In First Out)** 구조로 이루어지는 자료구조이다.

### 큐 용어

#### 위치

- front : 가장 먼저 들어온 데이터
- back : 가장 마지막 데이터

#### 연산

- push : 데이터 삽입
- pop : 데이터 삭제

# ✅ 문제 11 — 백준 1874 (스택 수열)

## 핵심 아이디어

스택을 이용해 입력된 수열을 만들 수 있는지 확인한다.

## 정답 코드
```cpp
#include <iostream>
#include <stack>
using namespace std;

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N;
    cin >> N;

    stack<int> st;

    int num = 1;
    bool result = true;

    string answer;

    for(int i = 0; i < N; i++) {

        int target;
        cin >> target;

        while(num <= target) {
            st.push(num++);
            answer += "+\n";
        }

        if(st.top() == target) {
            st.pop();
            answer += "-\n";
        }
        else {
            result = false;
            break;
        }
    }

    if(result)
        cout << answer;
    else
        cout << "NO";
}
```

# ✅ 문제 12 — 백준 17298 (오큰수)

## 핵심 아이디어

스택에 인덱스를 저장하면서 오른쪽 큰 수 탐색

## 정답 코드
```cpp
#include <iostream>
#include <stack>
using namespace std;

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N;
    cin >> N;

    int A[1000000];
    int result[1000000];

    for(int i = 0; i < N; i++)
        cin >> A[i];

    stack<int> st;

    st.push(0);

    for(int i = 1; i < N; i++) {

        while(!st.empty() && A[st.top()] < A[i]) {
            result[st.top()] = A[i];
            st.pop();
        }

        st.push(i);
    }

    while(!st.empty()) {
        result[st.top()] = -1;
        st.pop();
    }

    for(int i = 0; i < N; i++)
        cout << result[i] << " ";
}
```

# ✅ 문제 13 — 백준 2164 (카드2)

## 핵심 아이디어

큐를 이용해 앞 카드 제거 후 다음 카드 뒤로 이동

## 정답 코드
```cpp
#include <iostream>
#include <queue>
using namespace std;

int main() {

    int N;
    cin >> N;

    queue<int> q;

    for(int i = 1; i <= N; i++)
        q.push(i);

    while(q.size() > 1) {

        q.pop();

        q.push(q.front());
        q.pop();
    }

    cout << q.front();
}
```

# 힙(Heap)

힙은 완전 이진 트리 기반 자료구조로 우선순위가 높은 데이터가 먼저 나오는 구조이다.

- 최소 힙 : 작은 값 우선
- 최대 힙 : 큰 값 우선

C++에서는 `priority_queue`로 구현할 수 있다.

# ✅ 문제 14 — 백준 11286 (절댓값 힙)

## 핵심 아이디어

절댓값 기준 정렬 후 값 비교

## 정답 코드
```cpp
#include <iostream>
#include <queue>
#include <cmath>
using namespace std;

struct cmp {

    bool operator()(int a, int b) {

        if(abs(a) == abs(b))
            return a > b;

        return abs(a) > abs(b);
    }
};

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N;
    cin >> N;

    priority_queue<int, vector<int>, cmp> pq;

    for(int i = 0; i < N; i++) {

        int x;
        cin >> x;

        if(x == 0) {

            if(pq.empty())
                cout << 0 << "\n";
            else {
                cout << pq.top() << "\n";
                pq.pop();
            }
        }
        else
            pq.push(x);
    }
}
```
