## 3. 톰캣 설치

[톰캣](https://tomcat.apache.org) 설치 링크

### 톰켓 설치 확인
```bash
apache-tomcat-9.0.84 파일 위치로 이동

cd ./apache-tomcat-9.0.84/bin

톰캣 실행
./startup.sh

톰캣 종료
./shutdown.sh
```
