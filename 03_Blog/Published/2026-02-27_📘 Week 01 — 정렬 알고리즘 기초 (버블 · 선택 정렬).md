# 📘 Week 01 — 정렬 알고리즘 기초 (버블 · 선택 정렬)

# 버블 정렬 (Bubble Sort)

## 버블 정렬의 핵심 이론

버블 정렬은 두 인접한 데이터를 비교하여 정렬하는 방법이다.  
구현은 간단하지만 시간 복잡도가 **O(N²)** 으로 다른 정렬 알고리즘에 비해 느리다.

---

## 버블 정렬 과정

1. 비교 연산이 필요한 루프 범위를 설정한다.
2. 인접한 데이터 값을 비교한다.
3. swap 조건에 부합하면 swap 연산을 수행한다.
4. 루프 범위가 끝날 때까지 2~3 과정을 반복한다.
5. 정렬이 완료된 영역은 다음 반복에서 제외한다.
6. 비교 대상이 없을 때까지 반복한다.

# ✅ 문제 15 — 백준 2750 (수 정렬하기)

## 핵심 아이디어

버블 정렬을 직접 구현하여 오름차순 정렬한다.

## 정답 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    int N;
    cin >> N;

    int A[1000];

    for(int i = 0; i < N; i++)
        cin >> A[i];

    for(int i = 0; i < N - 1; i++) {

        for(int j = 0; j < N - 1 - i; j++) {

            if(A[j] > A[j + 1]) {

                int temp = A[j];
                A[j] = A[j + 1];
                A[j + 1] = temp;
            }
        }
    }

    for(int i = 0; i < N; i++)
        cout << A[i] << "\n";
}
```

# ✅ 문제 16 — 백준 1377 (버블 소트)

## 핵심 아이디어

정렬 전 위치와 정렬 후 위치의 차이를 이용한다.

버블 정렬에서 한 번의 반복마다 최대 한 칸만 이동할 수 있으므로  
`최대 이동 거리 + 1` 이 정답이 된다.

## 정답 코드

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

pair<int,int> A[500000];

int main() {

    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N;
    cin >> N;

    for(int i = 0; i < N; i++) {

        cin >> A[i].first;
        A[i].second = i;
    }

    sort(A, A + N);

    int maxMove = 0;

    for(int i = 0; i < N; i++) {

        if(maxMove < A[i].second - i)
            maxMove = A[i].second - i;
    }

    cout << maxMove + 1;
}
```

# 선택 정렬 (Selection Sort)

## 선택 정렬의 핵심 이론

선택 정렬은 남은 데이터 중 최솟값 또는 최댓값을 찾아 정렬하는 방식이다.

구현 방식이 비교적 단순하지만 시간 복잡도가 **O(N²)** 으로 비효율적이기 때문에 실제 개발에서는 거의 사용하지 않는다.

---

## 선택 정렬 과정

1. 남은 정렬 부분에서 최솟값 또는 최댓값을 찾는다.
2. 남은 정렬 부분의 가장 앞 데이터와 선택된 데이터를 swap한다.
3. index를 증가시켜 정렬 영역을 확장한다.
4. 남은 데이터가 없을 때까지 반복한다.

# ✅ 문제 17 — 백준 1427 (소트인사이드)

## 핵심 아이디어

숫자를 자리수 단위로 나누어 내림차순 정렬한다.

## 정답 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    string str;
    cin >> str;

    int N = str.length();

    for(int i = 0; i < N; i++) {

        int maxIdx = i;

        for(int j = i + 1; j < N; j++) {

            if(str[j] > str[maxIdx])
                maxIdx = j;
        }

        char temp = str[i];
        str[i] = str[maxIdx];
        str[maxIdx] = temp;
    }

    cout << str;
}
```
