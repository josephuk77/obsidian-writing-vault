`00_Inbox/`는 아직 분류하지 않은 아이디어, 빠른 메모, 임시 원문을 모으는 공간입니다.

## 사용 규칙

- 빠르게 캡처해야 하는 내용은 먼저 이곳에 둡니다.
- 프로젝트와 관련된 원문은 정리 후 해당 프로젝트의 `10_RAW_SOURCE/`로 옮깁니다.
- 블로그로 발전시킬 글감은 `03_Blog/`로 옮깁니다.
- 학습 노트로 정리된 내용은 `02_Study/`로 옮깁니다.

## 정리 기준

- 원문 보존이 필요한 자료는 [[raw-source]] 템플릿을 사용합니다.
- 출처가 있는 자료는 링크와 캡처 날짜를 남깁니다.
- 한 달에 한 번 이상 오래된 Inbox 문서를 분류합니다.
