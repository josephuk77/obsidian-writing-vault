## 글의 목적

PT 면접에서 IoT를 설명할 때 단순히 "사물이 인터넷에 연결되는 기술"이라고만 말하면 얕게 들릴 수 있다.

이 글의 목적은 IoT를 센서, 네트워크, 데이터, 클라우드, 엣지 컴퓨팅, AI와 연결해서 설명할 수 있게 만드는 것이다.

특히 인공지능과 함께 발표할 주제이므로, IoT가 데이터를 수집하고 AI가 그 데이터를 분석한다는 연결 구조를 명확하게 잡는다.

## 핵심 결론

IoT는 사물에 센서와 통신 기능을 붙여 현실 세계의 데이터를 수집하고, 네트워크를 통해 다른 시스템과 연결하는 기술이다.

IoT의 핵심 가치는 단순 연결이 아니라 데이터 수집과 자동화에 있다. 스마트홈, 스마트팩토리, 헬스케어, 물류, 스마트시티처럼 현실 세계에서 발생하는 데이터를 수집하고 분석해 더 빠른 판단과 제어를 가능하게 만든다.

AI와 결합하면 IoT는 더 강력해진다. IoT가 현실 데이터를 모으고, AI가 그 데이터를 분석해 이상 감지, 예측, 자동 제어를 수행할 수 있기 때문이다.

핵심 문장:

> IoT는 현실 세계의 데이터를 디지털 세계로 가져오고, AI는 그 데이터를 판단과 자동화로 바꾸는 역할을 한다.

## 개념

### IoT란 무엇인가

IoT는 Internet of Things, 즉 사물인터넷이다.

스마트폰이나 컴퓨터뿐 아니라 조명, 냉장고, 자동차, 공장 설비, CCTV, 웨어러블 기기 같은 사물이 인터넷에 연결되어 데이터를 주고받는 구조를 말한다.

핵심은 세 가지다.

- 현실 세계의 데이터를 센서로 수집한다.
- 네트워크를 통해 데이터를 전달한다.
- 수집한 데이터를 분석하거나 자동 제어에 활용한다.

### IoT의 기본 흐름

```text
센서가 데이터 수집
-> 네트워크로 데이터 전송
-> 서버/클라우드/엣지에서 데이터 처리
-> 분석 결과 생성
-> 사용자 알림 또는 기기 자동 제어
```

### IoT와 AI의 차이

IoT는 데이터를 모으고 연결하는 기술에 가깝다.

AI는 그 데이터를 분석하고 판단하는 기술이다.

그래서 둘을 함께 설명하면 다음처럼 정리할 수 있다.

```text
IoT = 현실 데이터를 수집하는 기술
AI = 수집된 데이터를 분석하고 판단하는 기술
AIoT = IoT + AI
```

면접에서 사용할 문장:

> IoT가 현실 세계의 데이터를 수집하는 역할을 한다면, AI는 그 데이터를 분석해 예측과 자동 제어를 가능하게 하는 역할을 합니다.

## 장점

IoT의 장점은 단순히 사물을 인터넷에 연결하는 데서 끝나지 않는다.

- 실시간 모니터링이 가능하다.
- 사람이 직접 확인하지 않아도 데이터를 자동으로 수집할 수 있다.
- 이상 상황을 빠르게 감지할 수 있다.
- 반복적인 제어를 자동화할 수 있다.
- 에너지, 비용, 시간을 줄일 수 있다.
- 데이터를 기반으로 더 정확한 의사결정을 할 수 있다.

## 단점

IoT는 연결되는 기기가 많아질수록 관리해야 할 문제도 늘어난다.

- 보안 취약점이 늘어난다.
- 개인정보와 생활 패턴 데이터가 노출될 수 있다.
- 기기마다 통신 방식과 표준이 달라 연동이 어려울 수 있다.
- 네트워크 장애가 발생하면 서비스가 멈출 수 있다.
- 설치와 유지보수 비용이 발생한다.
- 수집되는 데이터가 너무 많아 저장과 분석 부담이 커질 수 있다.

## 해결방안

IoT를 안전하게 운영하려면 기기, 네트워크, 데이터, 사용자 권한을 함께 관리해야 한다.

- 기본 비밀번호를 바꾸고 강한 인증을 적용한다.
- 기기 펌웨어를 주기적으로 업데이트한다.
- 불필요한 네트워크 접근을 차단한다.
- 중요한 데이터는 암호화한다.
- 사용자 권한을 최소한으로 부여한다.
- IoT 기기와 일반 업무망을 분리한다.
- 장애 상황을 대비해 수동 제어 방식도 남겨둔다.
- 데이터 수집 목적과 보관 기간을 명확히 정한다.

핵심 문장:

> IoT는 많이 연결할수록 편리해지지만, 그만큼 보안과 데이터 관리 기준을 먼저 세워야 합니다.

## 활용사례

### 스마트홈

조명, 냉난방, 도어락, CCTV, 가전제품을 연결해 사용자의 생활 패턴에 맞게 자동 제어한다.

예시:

- 외출 시 조명과 냉난방 자동 종료
- 문 열림 감지 후 스마트폰 알림
- 실내 온도에 따른 에어컨 자동 제어

### 스마트팩토리

공장 설비와 생산 데이터를 연결해 품질, 생산성, 유지보수를 개선한다.

예시:

- 설비 진동 데이터로 고장 예측
- 생산 라인 데이터로 불량률 분석
- 작업자 안전 장비와 위치 데이터 모니터링

### 헬스케어

웨어러블 기기나 의료 센서로 건강 데이터를 수집하고 분석한다.

예시:

- 심박수, 수면, 산소포화도 모니터링
- 고령자 낙상 감지
- 원격 환자 상태 확인

### 물류와 유통

제품 위치, 온도, 재고 상태를 실시간으로 추적한다.

예시:

- 냉장 물류 온도 모니터링
- 배송 차량 위치 추적
- 매장 재고 자동 파악

### 스마트시티

도시의 교통, 안전, 환경, 에너지 데이터를 연결해 도시 운영을 개선한다.

예시:

- 교통량 기반 신호 제어
- 스마트 가로등
- 미세먼지 센서
- 침수 감지 센서
- CCTV 기반 안전 관제

### 엣지 AI와 IoT가 결합되고 있다

## PT 발표 흐름

## 예상 질문과 답변

### IoT란 무엇인가요?

IoT는 사물에 센서와 통신 기능을 붙여 데이터를 수집하고 네트워크로 연결하는 기술입니다. 스마트홈, 스마트팩토리, 헬스케어, 스마트시티 등에 활용됩니다.

### IoT와 AI는 어떤 관계인가요?

IoT는 현실 세계의 데이터를 수집하고, AI는 그 데이터를 분석합니다. 둘이 결합되면 이상 감지, 예측, 자동 제어가 가능해집니다.

### IoT의 가장 큰 장점은 무엇인가요?

실시간 모니터링과 자동화입니다. 사람이 직접 확인하지 않아도 데이터를 수집하고, 조건에 따라 기기가 자동으로 동작할 수 있습니다.

### IoT의 가장 큰 문제는 무엇인가요?

보안과 개인정보 문제입니다. 연결된 기기가 많아질수록 공격 지점도 늘어나고, 생활 패턴이나 위치 같은 민감한 데이터가 노출될 수 있습니다.

### IoT 보안은 어떻게 해결할 수 있나요?

기기 인증, 암호화, 펌웨어 업데이트, 접근 권한 관리, 네트워크 분리, 제로 트러스트 같은 보안 체계를 적용해야 합니다.

## 정리

IoT는 사물을 인터넷에 연결하는 기술이지만, 핵심은 연결 그 자체가 아니라 현실 세계의 데이터를 수집하고 활용하는 데 있다.

AI 시대에는 IoT의 중요성이 더 커진다. AI가 좋은 판단을 하려면 현실 데이터가 필요하고, IoT는 그 데이터를 수집하는 입구 역할을 하기 때문이다.

면접에서 사용할 결론:

> IoT는 현실 세계의 데이터를 수집하고 연결하는 기술입니다. 스마트홈, 스마트팩토리, 헬스케어, 스마트시티처럼 다양한 분야에서 실시간 모니터링과 자동화를 가능하게 합니다. 다만 연결되는 기기가 늘어날수록 보안과 개인정보 문제가 커지기 때문에, 인증, 암호화, 업데이트, 네트워크 분리 같은 관리 체계가 함께 필요합니다. 특히 AI와 결합하면 IoT가 수집한 데이터를 바탕으로 예측과 자동 제어까지 가능해진다는 점에서 앞으로 더 중요한 기술이라고 생각합니다.
