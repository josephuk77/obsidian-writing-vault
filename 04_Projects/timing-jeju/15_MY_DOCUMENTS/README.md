tags:

사용자가 프로젝트를 진행하면서 직접 작성한 문서를 넣는 공간입니다.

## 역할

- 아이디어 초안
- 기획 메모
- 회고
- 리뷰
- 직접 작성한 설명 문서

## 하위 폴더

| 폴더 | 역할 |
| --- | --- |
| `drafts/` | 아직 정리 중인 초안 |
| `notes/` | 직접 작성한 프로젝트 메모 |
| `reviews/` | 회고, 검토, 피드백 |

## 규칙

- 사용자가 직접 쓴 문서는 우선 이곳에 보관합니다.
- 프로젝트 실행에 필요한 확정 정보는 `20_CORE/`로 요약해 옮깁니다.
- AI가 만든 정리본과 구분하기 위해 원문 성격의 사용자 문서는 그대로 보존합니다.
