# 05_Resources

---
tags:
  - type/project
  - status/active
---

`05_Resources/`는 장기 보관할 참고자료, 링크, 인용, 학습 리소스를 주제별로 관리하는 공간입니다.

## 하위 폴더

| 폴더 | 역할 |
| --- | --- |
| `backend/` | Spring, DB, Redis, API, 서버 설계 |
| `ai/` | AI 도구, LLM, 프롬프트, 자동화 |
| `career/` | 취업, 면접, 포트폴리오, 커리어 전략 |
| `education/` | 강의, 과정, 교육 자료 |
| `writing/` | 글쓰기, 블로그, 콘텐츠 참고자료 |

## 사용 규칙

- 원문 보존이 중요한 자료는 [[raw-source]] 템플릿을 사용합니다.
- 프로젝트 전용 자료는 가능하면 해당 프로젝트의 `10_RAW_SOURCE/references/`에 둡니다.
- 여러 프로젝트에서 쓰는 범용 자료만 이곳에 보관합니다.
