수정하지 않는 원본 저장소입니다.

## 하위 폴더

| 폴더 | 역할 |
| --- | --- |
| `articles/` | 웹 글, 블로그, 문서 원문 |
| `assets/` | 이미지, 스크린샷, 첨부자료 |
| `inbox/` | 아직 분류하지 않은 원문 |
| `notes/` | 사용자가 붙여넣은 원본 메모 |
| `papers/` | 논문, PDF, 긴 원문 자료 |

## 규칙

- 이 폴더의 파일은 원본 보존용입니다.
- AI는 이 폴더의 원문을 수정하거나 삭제하지 않습니다.
- 정리본은 `../wiki/`에 작성합니다.
- 원문을 추가하면 [[_MANIFEST]]에 기록합니다.
