AI가 만든 산출물을 보관하는 공간입니다.

## 하위 폴더

| 폴더 | 역할 |
| --- | --- |
| `prompts/` | 재사용 가능한 프롬프트 |
| `reports/` | 분석 리포트, 리뷰 결과 |
| `specs/` | 요구사항, 기획, 구현 스펙 |

## 규칙

- 프로젝트 전용 산출물은 가능하면 `04_Projects/{project}/90_OUTPUTS/`에 둡니다.
- 여러 프로젝트에서 재사용할 수 있는 산출물만 이곳에 둡니다.
