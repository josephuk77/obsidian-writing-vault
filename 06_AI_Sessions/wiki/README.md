AI가 Raw Source를 바탕으로 정리하고 연결하는 전역 업무 지식 공간입니다.

## 하위 폴더

| 폴더 | 역할 |
| --- | --- |
| `concepts/` | 반복해서 쓰는 개념 정리 |
| `decisions/` | 전역 의사결정과 이유 |
| `entities/` | 사람, 회사, 도구, 시스템 정보 |
| `errors/` | 문제와 해결 기록 |
| `playbooks/` | 반복 업무 절차 |
| `projects/` | 여러 프로젝트에 걸친 요약 |
| `sources/` | 원본 자료의 정리본 |
| `synthesis/` | 여러 자료를 종합한 결론 |

## 규칙

- Raw Source를 근거로 정리합니다.
- 근거가 불확실하면 추정이라고 표시합니다.
- 프로젝트 전용 내용은 `04_Projects/{project}/20_CORE/` 또는 `20_WIKI/`에 연결합니다.
